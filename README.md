# 临沂旅游推荐 APP · 임기 여행 추천 APP

> 使用 Flutter 开发的中国家乡旅行推荐应用
> Flutter로 개발한 중국 고향 여행 추천 앱

---

## 项目介绍 · 프로젝트 소개

本项目是期末团队作业，由本人独立负责临沂页面的设计与开发。
이 프로젝트는 기말 팀 과제로, 임기 페이지의 설계와 개발을 본인이 직접 담당했습니다.

临沂是我的家乡，我对这里的景点和美食非常熟悉，所以选择用 Flutter 来展示家乡的魅力。
임기는 제 고향으로, 이곳의 관광지와 음식을 잘 알고 있어 Flutter로 고향의 매력을 소개하게 되었습니다.

所有页面内容均采用**中韩双语**显示。
모든 페이지 내용은 **중한 이중언어**로 표시됩니다.

## 主要功能 · 주요 기능

| # | 功能 / 기능 | 说明 / 설명 |
|:---:|---|---|
| 1 | 家乡介绍 · 고향 소개 | 城市概况、基本数据、荣誉称号、发展实力、人文底蕴 / 도시 개황, 기본 데이터, 명예 칭호, 발전 역량, 인문 역사 |
| 2 | 代表旅游景点 · 대표 관광지 | 4个景点卡片，含图片、评分、中韩双语介绍 / 4개 관광지 카드, 이미지·평점·중한 이중언어 소개 포함 |
| 3 | 代表美食 · 대표 음식 | 4种美食卡片，含图片、评分、中韩双语介绍 / 4가지 음식 카드, 이미지·평점·중한 이중언어 소개 포함 |
| 4 | 推荐旅行路线 · 추천 여행 코스 | 1日游时间轴，6个行程节点 / 당일치기 타임라인, 6개 일정 노드 |
| 5 | AI使用内容 · AI 활용 내용 | 说明本页面中AI辅助完成的内容 / 본 페이지에서 AI가 보조한 내용 설명 |

## AI 使用内容 · AI 활용 내용

> 本项目以本人独立开发为主，AI 主要用于辅助翻译和查阅资料。
> 본 프로젝트는 본인이 직접 개발한 것을 위주로 하며, AI는 주로 번역 보조와 자료 조회에 사용했습니다.

| 韩文翻译辅助 / 한국어 번역 보조 | 景点、美食的韩文介绍翻译参考 / 관광지·음식 한국어 소개 번역 참고 |
