# 临沂旅游推荐 APP · 임기 여행 추천 APP

> 使用 Flutter 开发的中国家乡旅行推荐应用
> Flutter로 개발한 중국 고향 여행 추천 앱

---

## 项目介绍 · 프로젝트 소개

本项目是期末团队作业，使用 Flutter 开发，向用户介绍中国山东省临沂市的旅游资源。
본 프로젝트는 기말 팀 과제로, Flutter를 사용하여 중국 산둥성 임기시의 관광 정보를 소개합니다.

所有页面内容均采用**中韩双语**显示。
모든 페이지 내용은 **중한 이중언어**로 표시됩니다.

---

## 主要功能 · 주요 기능

| # | 功能 / 기능 | 说明 / 설명 |
|:---:|---|---|
| 1 | 家乡介绍 · 고향 소개 | 城市概况、基本数据、荣誉称号、发展实力、人文底蕴 / 도시 개황, 기본 데이터, 명예 칭호, 발전 역량, 인문 역사 |

| 4 | 推荐旅行路线 · 추천 여행 코스 | 1日游时间轴，6个行程节点 / 당일치기 타임라인, 6개 일정 노드 |
| 5 | AI使用内容 · AI 활용 내용 | 说明本页面中AI辅助完成的内容 / 본 페이지에서 AI가 보조한 내용 설명 |
